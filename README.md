# KEE_Skuska_Otazky
Otázky na testík zo skúšky KEE

## Funkcie
- Rozdelenie otázok do 4 tematických testovacích okruhov (vrátane otvorených otázok zameraných na praktické príklady a definície).
- Okamžité vyhodnotenie správnosti odpovede s podrobným teoretickým odôvodnením (vysvetlením).
- Responzívny dizajn (funguje na počítači aj smartfóne).
